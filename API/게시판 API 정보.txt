[게시글 API 정보]

1. 게시글 등록
URL : /board/write
TYPE : POST 
파라미터 : 
{
	"title" : "글 제목", 
	"content" : "글 내용"
}

응답 : 
{
	"errorCode" : 0 ~ 2, // 에러 코드 값 0이 정상적으로 처리된 경우 입니다.
	"msg" : "처리 결과 메시지" // 에러나 성공에 대한 결과 메시지 입니다.
}




2. 게시글 수정
URL : /board/modify
TYPE : POST
파라미터 : 
{
	"boardId" : 1, // 게시글 번호입니다.
	"title" : "글 제목", 
	"content" : "글 내용"
}

응답 : 
{
	"errorCode" : 0 ~ 4, // 에러 코드 값 0이 정상적으로 처리된 경우 입니다.
	"msg" : "처리 결과 메시지" // 에러나 성공에 대한 결과 메시지입니다.
}




3. 게시글 삭제
URL : /board/delete
TYPE : POST
파라미터 : 
{
	"boardId" : 1 // 게시글 번호입니다.
}

응답 : 
{
	"errorCode" : 0 ~ 3, // 에러 코드 값 0이 정상적으로 처리된 경우 입니다.
	"msg" : "처리 결과 메시지" // 에러나 성공에 대한 결과 메시지입니다.
}
